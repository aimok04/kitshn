Создан с использованием <b>Jetpack Compose</b> и <b>Material You</b> для обеспечения современного, интуитивно понятного и адаптивного пользовательского интерфейса.

<b>Поддерживаемые функции:</b>
<ul>
    <li>Динамический главный экран с предложениями</li>
    <li>Рецепты</li>
    <ul>
        <li>Добавление, импорт, редактирование и удаление</li>
        <ul>
            <li>Инструмент для комбинирования шагов</li>
            <li>Получатель для импорта рецептов по URL</li>
        </ul>
        <li>Расширенный поиск</li>
        <li>Режим готовки</li>
        <li>Инструмент распределения ингредиентов по шагам</li>
        <li>Ссылки на приложения</li>
        <li>Поддержка Markdown</li>
    </ul>
    <li>План питания</li>
    <ul>
        <li>Обзор</li>
        <li>Добавление, редактирование и удаление</li>
    </ul>
    <li>Список покупок</li>
    <ul>
        <li>Добавление и удаление</li>
        <li>Категории и супермаркеты</li>
        <li>Режим покупок</li>
        <li>Офлайн-режим</li>
        <li>Авто-синхронизация</li>
    </ul>
    <li>Кулинарные книги</li>
    <ul>
        <li>Добавление и удаление</li>
        <li>Функция избранного</li>
    </ul>
    <li>UI</li>
    <ul>
        <li>Создан с использованием Jetpack Compose</li>
        <li>Динамические цвета Material You</li>
        <li>Адаптивные макеты с двумя панелями для планшетов и складных устройств</li>
    </ul>
</ul>

Обратите внимание, что приложение в настоящее время находится на альфа-этапе. Вы можете открывать проблемы на Github для сообщений об ошибках, отзывов и идей по функциям.
